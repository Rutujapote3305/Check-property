# Subscription & Payment Selector
An interactive form project that demonstrates how to handle user input through checkboxes and radio buttons using the JavaScript `.checked` property.

## Features
- **Subscription Validation:** Verifies if the "Subscribe" box is checked before processing.
- **Payment Detection:** Identifies the selected radio button (Visa, MasterCard, PayPal).
- **Dynamic Messaging:** Provides instant text feedback to the user based on their specific selection.
- **Interactive UI:** Hover effects and active states for a better user experience.

## Technical Stack
- **HTML5:** Input types (checkbox and radio).
- **CSS3:** Deep purple theme with a centered card layout and drop shadows.
- **JavaScript:** Boolean logic and conditional rendering.

## Learning Highlights
- Implementing the `.checked` property for boolean input validation.
- Using conditional logic to provide custom feedback to users.
- Managing multiple radio button inputs within a single form group.
